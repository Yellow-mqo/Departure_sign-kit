ダウンロードありがとうございます。
本パックはJR東日本管内などで見かける形の発車標を作成するためのKITです。

<内容物>
・モデルファイル(3種(内1種差分2個)+おまけパーツ2種)
・表示部の仮テクスチャを含むテクスチャ
・仮jsonファイル
・Readme
(フォルダ構成済み)


画像編集や、jsonの作成を前提としているため、ある程度pack作成の知識を必要とします。
が、仮テクスチャを用意しているのでそのままでも読み込むことは可能です。
(少し前に作ったものをkitように調整しただけなので、マッピングとテクスチャが少し雑です。)
また各所に.xcfファイルがありますが、GIMPをお持ちの方はご活用ください。

モニター表示部にはJavaScriptを使用し、内容を制御しています。
CustomNameに数字を入力するとそれに対応した表示に代わる仕様となっています。
詳しくは、このreadmeの最下部にある詳細を読むか、.jsファイルを直接ご確認ください。

また、モニター(特にLCDタイプ)に使用している画像サイズが大きいので読み込む際に重くなる場合があります。


デフォルトで照明モデルとして読み込みます。
設置の際、吊り下げ型なので天井(ブロックの下面)に向かって設置すればOKです。
(モデルから支柱を削除し、位置を調整すれば壁置きタイプも作成可)


ここまでお読みいただきありがとうございます。
自作発言だけ控えていただければ、何にご使用されても構いません。
ぜひご活用ください。

モデル変更画面上部右側にある"Custom Name"とツールチップが出るテキストボックスに
1桁の数字を入力すると、それに対応した表示に変わります。(数字を入力後、もう一度モデルを選択する必要があります。)
LEDタイプとLCDタイプとで使用が若干異なります。

LEDタイプ
0 : 調整中
1 : 通常
2 : 接近
3 : 通過

LCDタイプ
1 : 通常
2 : 在線
3 : 接近

日英交互表示や点滅表示についてはワールド時間を基準にtick数で表示を切り替えています。
どちらも.jsファイルに適宜コメントを記載してありますので、お好みで数値を変更できます。
