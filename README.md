# 発車標 KIT

本パックはJR東日本管内などで見かける形の発車標を作成するためのKITです。
  
## 🛠 内容物
- モデルファイル(3種(内1種差分2個)+おまけパーツ2種)
- 表示部の仮テクスチャを含むテクスチャ
 -仮jsonファイル
- Readme
(フォルダ構成済み)
  
## 🎮 実装について
デフォルトで照明モデルとして読み込みます。
設置の際、吊り下げ型なので天井(ブロックの下面)に向かって設置すればOKです。
(モデルから支柱を削除し、位置を調整すれば壁置きタイプも作成可)
  
## 📜 利用規約
- **✅ 可能**
  - 改造
  - 改造済の物の配布
  - 友人間の譲渡等
  - 他媒体での使用
- **🚫 禁止**
  - 未改造での再配布
  - 自作発言
- **📩 要相談**
  - 他媒体へ最適化したものの再配布
  
上記に気を付けていただければ、何にご使用されても構いません。  
また、不明な点ございましたら、都合の良い連絡手段でご連絡ください。  
RTM用に最適化されてはいますが、ぜひ多方面にご活用ください。
